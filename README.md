# Запуск
База данных: postgre 17
PHP: php 8.4
Laravel: ^12.0

## установка npm пакет
```npm 
npm install
```

## установка composer пакет
```
composer install
```

## миграция
```php
php artisan migrate
```

## Запуск проекта для laravel
```php
php artisan serve
```
## Запуск vue 
```npm
npm run dev
```
